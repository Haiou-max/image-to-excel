# 产品图片信息提取提示词

## 任务目标
从产品图片中提取信息，按以下 **7 列表头** 填入表格：

| 列 | 表头 | 类型 |
|----|------|------|
| A | 图片 | （图片本身） |
| B | 产品编号 | 必填 |
| C | 外箱容量 | 必填 |
| D | 客户意向价 | 必填 |
| E | 做货要求一 | 可空 |
| F | 箱数 | 必填 |
| G | 做货要求二 | 可空 |

---

## 一、标准图片格式（最高优先级）

大多数图片遵循以下格式：

```
上方：产品编号（X-X 格式）
下方一行：A件 × B PCS × ... × C¥
```

该行核心数据与字段的对应关系：

```
   20 件     ×   300 PCS   ×   2 ¥
    ↓              ↓             ↓
   箱数        外箱容量      客户意向价
（件 = 箱的单位）（PCS 前）   （¥ 前）
```

### 🔴 核心映射规则（必须严格遵守）
| 图片中的位置 | 对应字段 |
|------------|---------|
| `件` 前的数字（如 `20件`） | → **箱数**（填 `20`，不含「件」字） |
| `PCS` 前的数字 | → **外箱容量** |
| `¥` **前面**的数字（如 `2¥`） | → **客户意向价** |
| `X款`、`X个图案` 等计量词 | → **做货要求二** |
| `5×17`、`5×17×20` 等尺寸 | → **做货要求一** |

---

## 二、字段提取规则

### 1. 产品编号【必填】
- **位置**：图片上方
- **格式**：「几杠几」（如 `2-8`、`3-6`、`6-15`）
- **要求**：提取原始编号，不做改动

---

### 2. 外箱容量【必填】
- **来源**：`X PCS` 中 PCS 前面的数字
- **要求**：只保留数字

---

### 3. 客户意向价【必填】

- **位置**：`¥` 符号**前面**的数字（如 `2¥` → `2`）
- **要求**：提取数字，**小数点务必保留**

#### 🔴 强制纠错规则（手写小数点易漏）
手写价格中小数点可能**很小、很淡、或被连笔掩盖**，必须逐位仔细辨认。

| 误读 | 正确 |
|------|------|
| 39   | 3.9  |
| 55   | 5.5  |
| 105  | 10.5 |

#### 🔴 反思检查
- 外贸产品单价通常 **不超过 80 元**
- 若识别出的数字明显偏大，**很可能漏了小数点**
- 若识别超过 80 的数字，**必须检查是否漏了小数点**

---

### 4. 做货要求一（图片/尺寸相关）

**填入内容**：与「图片」直接相关的**尺寸规格**（二维或三维）。

✅ 典型示例：
- 二维尺寸：`5×17`、`7×9`、`15.5×20.5`
- 三维尺寸：`5×17×20`
- 单维尺寸：`7cm`

❌ 禁止放入：
- 「X件」「X款」「X付」「X组」「X个图案」等数量/计量词
- 颜色、款式、包装、版费、工艺等非尺寸信息

**留空条件**：完全没有尺寸信息时，此字段留空。

---

### 5. 箱数【必填】

#### 🔴 核心定义
- **位置**：图片中 `件` 字**前面**的数字
- **本质**：「件」是「箱」的计量单位，`20件 = 20 箱`
- **输出**：**只写数字，不写「件」字**（如 `20件` → 填 `20`）

#### 🔴 「件」字误读纠错（最关键）
手写「件」字的末尾笔画极易被误读成 `4`，导致箱数末尾多出一个 `4`：

| 误读  | 正确 |
|-------|------|
| 504   | 50   |
| 24    | 2    |
| 74    | 7    |

**强制检查流程**：
1. 提取箱数后，**检查末尾是否是 `4`**
2. 若末尾是 `4`，大概率这个 `4` 其实是「件」字，**应该去掉**
3. 箱数单元格**只能出现纯数字**，绝不能包含「件」字本身
4. 手写「件」的单人旁「亻」容易误读成 `4`

#### ❌ 禁止轻易标 `[待确认]`
标准格式下「件」字一定存在。只有图片极度模糊才能标待确认，否则必须逐笔画识别。

---

### 6. 做货要求二（所有其他信息）

**原则**：除上述字段外的**所有信息全部放这里**，不要遗漏。

#### 典型内容
- **款数**：如 `6款`、`10款`
- **图案数/说明**：如 `6个图案`、`字母图案`、`如图`
- **厚度**：如 `3mm厚度`
- **颜色**：如 `古锡30`、`古铜`
- **包装**：如 `opp袋`、`白盒`、`开天窗`
- **工艺**：如 `烫金`、`磨砂`、`亮膜`、`哑膜`、`对裱`、`过胶`
- **其他**：版费、白芯、付数、双色、渐变色等

#### 🔴 颜色+数量格式
若文字下面直接跟数字（如「古锡」下面写「30」），表示该颜色要多少个。

**输出格式**：数字紧跟颜色 → `古锡30`

常见颜色名：**古锡、古青、古铜、古银、古金**。

#### 🔴 「个」字误读纠错
手写「个」字容易被误认成「付」。数量语境下（如「30个」）不要写成「30付」，除非图片中明确是「付」。

#### 多项信息分隔
用中文逗号「，」分隔，例如：
`3mm厚度，6个图案，opp袋`

## 三、字段归属速查表

| 图片中看到 | 归属字段 | 填入值 |
|-----------|---------|--------|
| `2-7`（X-X 编号） | 产品编号 | `2-7` |
| `20件` | **箱数** | `20` |
| `300 PCS` | 外箱容量 | `300` |
| `2¥` | 客户意向价 | `2` |
| `5×17` | 做货要求一 | `5×17` |
| `5×17×20` | 做货要求一 | `5×17×20` |
| `6款` | 做货要求二 | `6款` |
| `6个图案` | 做货要求二 | `6个图案` |
| `3mm厚度` | 做货要求二 | `3mm厚度` |
| `opp袋`、`白盒` | 做货要求二 | 原文 |
| `古锡` 下写 `30` | 做货要求二 | `古锡30` |

## 四、手写常见误读纠错表

工厂做货术语词库，用以下纠正识别结果：

| 正确术语 | 易误读为 | 说明 |
|---------|---------|------|
| 字母 / 字母图案 | 蝴 / 蝴蝶 | — |
| 不要 / 不要 XX | 硬 / 硬件 | 手写「不」易看成「硬」 |
| 颜色 / 颜色亮 / 颜色深 | 瓷 / 瓷亮 | 手写「颜色」二字易误认 |
| 图案 / 组图案 | 国案 / 国家 | — |
| 如图 | 超圆 / 起圆 | 常见标注用语 |
| 件 | 4 | 末尾「亻」旁易误读 |
| 个 | 付 | 数量语境优先识别为「个」 |

### 常见做货术语（候选词库）
`如图` `单色` `双色` `渐变色` `不要渐变色` `字母` `数字` `图案` `颜色亮` `颜色深` `白芯` `白盒` `opp袋` `烫金` `开天窗` `磨砂` `亮膜` `哑膜` `对裱` `过胶`

> **兜底规则**：若识别出的词不通顺且不在词库中，尝试用**最接近的常见术语替换**。

## 五、手写数字辨认

手写 `6`、`8`、`3` 笔画相似易混淆，逐笔画核对：

| 数字 | 特征 |
|------|------|
| 8    | 两个圈 |
| 6    | 上开口，下封闭 |
| 3    | 两个弧向右开口 |

**易错示例**：
- `5.8` 不要识别成 `5.6`
- `5.3` 不要识别成 `5.8`

## 六、提取输出示例

**图片内容**：
```
产品编号：2-7
规格行：20件 × 300 PCS × 2¥
其他标注：5×17、3mm厚度、6个图案、opp袋
```

**输出行**：
| 产品编号 | 外箱容量 | 客户意向价 | 做货要求一 | 箱数 | 做货要求二 |
|---------|---------|-----------|-----------|------|-----------|
| 2-7 | 300 | 2 | 5×17 | 20 | 3mm厚度，6个图案，opp袋 |

## 七、提取前自检清单

- [ ] 产品编号格式是否为 `X-X`？
- [ ] **箱数是否取自「件」前的数字？（只填数字，不含「件」字）**
- [ ] **箱数末尾是否是 `4`？若是，是否确认不是「件」字误读？**
- [ ] **客户意向价是否取自 `¥` 前的数字？**
- [ ] 客户意向价是否 > 80？若是，是否检查了漏小数点？
- [ ] 做货要求一是否只含尺寸（`X×Y` 或 `X×Y×Z`）？
- [ ] `X款`、`X个图案` 这类计量词是否放到了做货要求二（不是做货要求一）？
- [ ] 做货要求二是否包含了款数、图案、厚度、颜色、包装、工艺等所有其他信息？
- [ ] 手写不通顺的词是否用常见术语词库纠正过？
